Assignment 2- Methods of String and Array

I.Methods of Array

1.toUpperCase()

Convert all letters in the uppercase
syntax- string.toUpperCase()

2.toLowerCase()

Convert all letters in the lowercase
syntax- string.toLowerCase()

3.charAt()

Return the given character of a string.
syntax- string.charAt(index)

4.charCodeAt()

Return the Unicode of the givens character in a string
syntax -string.charCodeAt(index)

5.concat()

Join two strings
Syntax - string.concat(string1, string2, ..., stringX)

6.endsWith()

This method checks the string is end with given character.
if it end with given character then written true otherwise false.
Syntax- string.endsWith(searchvalue, length)

7.fromCharCode()

Convert a Unicode number into a character
Syntax -String.fromCharCode(n1, n2, ..., nX)

8. includes()

Check if a string includes it or not if yes it print true else false
Syntax - string.includes(searchvalue, start)

9.substring()

Extract characters from a string
Syntax -string.substring(start, end)

10. slice()

Extract parts of a string
syntax -string.slice(start, end)

11.replace()

replace strings
Syntax- string.replace(searchvalue, newvalue)

II. Methods of Array

1.concat()

Join two arrays.
Syntax- array1.concat(array2, array3, ..., arrayX)

2.copyWithin()

Copy the first two array elements to the last two array elements.
Syntax- array.copyWithin(target, start, end)

3. fill()

Fill all the array elements with a static value
Syntax -array.fill(value, start, end)

4. lenght

Return the length of an array
Syntax -array.length

5. isarray()

Check whether an object is an array
Syntax -Array.isArray(obj)

6.indexOf()

Search an array for the item
Syntax -array.indexOf(item, start)

7.tostring()

The JavaScript method toString() converts an array to a string of (comma separated) array values.
Syntax-array.toString()

8. sort()

Sort an array
Syntax-array.sort(compareFunction)

9.splice()

Add items to the array
Syntax-array.splice(index, howmany, item1, ....., itemX)

10.reverse()

Reverse the order of the elements in an array.
Syntax-array.reverse()
